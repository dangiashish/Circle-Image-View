#### Gradle
 ```gradle
 allprojects {
		repositories {
			maven { url 'https://jitpack.io' }
		}
	}
 ```
 
 ### Add dependency
 
 ```Dependency
 dependencies {
	        implementation 'com.github.dangiashish:Circle-Image-View:1.0.2'
	}
 ```
